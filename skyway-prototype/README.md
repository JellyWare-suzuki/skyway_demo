# SkyWay Prototype

SkyWay P2P hands-on sample. Open index.html directly in a browser or serve the directory for local testing. Replace `appId` and `secretKey` in `script.js` with your own credentials before real use.

## Local run
npx serve .

Or:

python3 -m http.server 8000

## Deploy
Static-friendly layout ready for Netlify / Vercel / GitHub Pages. Deploy the `skyway-prototype` directory as-is.
